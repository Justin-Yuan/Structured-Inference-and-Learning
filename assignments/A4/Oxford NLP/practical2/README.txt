practical2 folder structure

- practical2.pdf contains assignment instructions from the Oxford NLP courses practical2 and model architectures.

- practical2.ipynb is a rough completion of the assignment, more of experimental purpose and it's deprecated.

- practical2-clean.ipynb is the refactored and cleaned completion. 

- models folder contains trained tensorflow models for the assignment. 


Notes:

The training result is bad, with a test accuracy at 54% (for 30000 epochs), only slightly above the majority class predictor at 53%.
